# Persona — [Projeto] — Squad [X]

> Artefato da Inception (Fase 1), pré-requisito do M1. Uma persona por arquétipo de usuário relevante — duplique a tabela para cada uma. Sem dados pessoais reais: a persona é uma representação fictícia baseada nos setores entrevistados.

| Campo | Resposta |
|---|---|
| **Nome fictício** | Apelido que ajude a equipe a lembrar dela (ex.: "Dona Lúcia, da assistência") |
| **Papel / cargo** | Função na instituição (sem citar pessoa real) |
| **Contexto de uso** | Onde, quando e com que recursos usa o sistema (computador, celular, internet do campus...) |
| **Objetivos** | O que essa pessoa quer alcançar com a solução |
| **Dores e frustrações** | O que hoje a atrapalha no processo atual |
| **Necessidades** | O que a solução precisa oferecer para ela |
| **Nível de familiaridade com tecnologia** | Baixo / Médio / Alto |
| **Frase típica** | Algo que essa pessoa diria (ajuda a humanizar) |

## Critério de pronto

- [ ] Cada persona vem de um setor/usuário realmente entrevistado, não inventado do zero.
- [ ] As dores listadas conversam com o **Problema** registrado em `visao-produto.md`.
- [ ] A persona principal coincide com o **Usuário principal** da visão.
