# Registro Semanal — Squad [X] — Semana [N]

> Preenchido pelo squad e revisado no checkpoint técnico.

| Campo | Conteúdo |
|---|---|
| **Squad** | Nome do squad e integrantes |
| **Semana** | Número da semana e fase atual |
| **Objetivo da semana** | Resultado esperado para o ciclo |
| **Entregas realizadas** | O que foi concluído, com evidência (link de commit, PR, protótipo) |
| **Pendências** | O que ficou aberto |
| **Impedimentos** | Bloqueios técnicos, institucionais ou de equipe → abrir Issue `impedimento` se durar mais de 1 dia |
| **Decisões tomadas** | Decisões de produto, arquitetura ou processo |
| **Próximos passos** | Prioridades para a próxima semana |
| **Apoio necessário** | O que precisa de instrutores, coordenação ou DTI |
